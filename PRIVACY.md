# Privacy Policy - RoBrowser

Last updated: 2026-04-18

RoBrowser is built to help users find Roblox servers faster. We designed it to work without collecting personal data.

## What RoBrowser Collects

- RoBrowser does not collect personal data.
- RoBrowser does not collect account credentials.
- RoBrowser does not collect payment data.
- RoBrowser does not track browsing outside Roblox game pages.

## What RoBrowser Processes

RoBrowser reads server metadata from Roblox APIs to show:

- server player counts
- server ping categories
- avatar headshots provided by Roblox thumbnail endpoints

This data is used only to render the extension UI in your browser session.

## Permissions

RoBrowser uses host permissions to access only these domains:

- `https://www.roblox.com/*`
- `https://games.roblox.com/*`
- `https://thumbnails.roblox.com/*`

RoBrowser does not use the `storage` permission.

## Data Sharing

RoBrowser does not sell user data and does not share personal data with third parties.
